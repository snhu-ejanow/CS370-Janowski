# CS370-Janowski
I was given the code that controlled the initial variables for the model, the maze it had to solve, and various helper functions that managed the memory recall and other complex aspects of the neural network. The code I created called these various helper functions during the execution of the neural network's training, initializing the environment state and looping through the agent's exploitation and exploration actions while the current game is active, feeding the data to the neural network and storing the episode in an experience replay, which was then used to update the model's values.
